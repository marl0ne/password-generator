#  Password Generator

##  Description

This is a Python command-line program that generates a secure random password based on user preferences. The user can choose the password length and select which character types to include.

##  Features

* Custom password length
* Option to include:

  * Uppercase letters
  * Lowercase letters
  * Numbers
  * Symbols
* Input validation (prevents invalid entries)
* Random password generation

##  How to Run

1. Make sure Python is installed on your system
2. Download or clone the repository
3. Run the program:

```bash
python password_generator.py
```
